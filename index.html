UinqueDeviceID and DeviceID are the device IDs for each set of data.
Latitude and Longitude are the latitude and longitude for the device.
StateOfCharge is the current battery life for the battery connected to the particle.
AwakeState is a 1 if the device is currently moving and a 0 if the device is sleeping.
Running is a 1 if the device has gotting a Start Up message from a user or this console.
AliveState should always be a 1 unless it has revieved a ForceTurnOff message from this console.
Timestamp is the timestamp of the data set.
ForceTurnOff will send a force turn off message to the deivce, this will set it to a permanent sleep state
StartUp will send a start up message to the device that will speed up the data transfer rate, this should be the same as the user option for starting a device.
